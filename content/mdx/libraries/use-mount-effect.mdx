마운트 직후 콜백 함수를 한 번만 실행하는 훅입니다. 마운트 로직을 명시적으로 분리하고 싶을 때 사용합니다.

# 구현

```ts
import { useEffect } from 'react';

export function useMountEffect(callback: () => void): void {
  useEffect(() => {
    callback();
  }, []);
}
```

---

# 사용법

## 1. 마운트 시 한 번만 실행

```tsx
function Component() {
  useMountEffect(() => {
    console.log('마운트됨');
    fetchInitialData();
  });

  return <div>...</div>;
}
```
