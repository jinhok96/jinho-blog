> 원문: [Build new features using built-in AI in Chrome](https://developer.chrome.com/blog/build-new-features-using-built-in-ai-in-chrome-io2026?hl=en) — Chrome Developers

TrAIlblazers 팀은 사용자가 글쓰기를 지원하는 블로그 편집기를 만들었습니다. 이들은 [Chrome 내장 AI](https://developer.chrome.com/docs/ai/built-in)를 사용하여 사용자 디바이스에서 직접 모델을 실행함으로써, 개발자는 비싼 클라우드 비용과 지연 시간을 피하면서 민감한 데이터를 로컬에 유지할 수 있습니다.

Google은 Build Awesome(이전 Eleventy)과 협력하여 토크에서 언급된 모든 AI 기능이 포함된 블로그 템플릿을 출시했습니다.

*   **비용 효율적:** 클라우드 추론 비용이 없으며, 모든 계산은 사용자의 지원 기기에서 이루어집니다.
*   **개인 정보 우선:** 민감한 데이터는 브라우저를 벗어나지 않습니다.
*   **오프라인 기능:** 모델이 다운로드되면 AI 기능은 인터넷 연결 없이 작동합니다.
*   **성능:** 하드웨어 가속을 통해 온디바이스 모델이 클라우드 속도에 필적하거나 때로는 능가할 수 있습니다.
*   **하이브리드 추론:** 폴리필과 **Firebase AI Logic**과 같은 도구를 사용하여 지원되지 않는 기기(모바일 등)에서는 클라우드로 대체하고 데스크톱에서는 네이티브를 유지할 수 있습니다.

# 최신 웹 앱을 위한 AI 기능

## Summarizer API

trAIlblazers 편집기는 [Summarizer API](https://developer.chrome.com/docs/ai/summarizer-api)를 사용하여 헤드라인과 SEO 친화적인 메타 설명을 자동으로 생성합니다.

**예시: 헤드라인 생성**

```
const blogPost = document.querySelector('.article-body').innerText;
const summarizer = await Summarizer.create({
  type: 'headline',
  sharedContext: 'Write headlines that make people want to read the blog post',
});

for await (const chunk of summarizer.summarizeStreaming(blogPost)) {
  headline.append(chunk);
}

```

## Prompt API (구조화된 출력 포함)

특정 데이터가 필요하신가요? [**JSON Schema**와 Prompt API](https://developer.chrome.com/docs/ai/structured-output-for-prompt-api)를 사용하면 AI가 예측 가능한 형식으로 결과를 반환하도록 할 수 있습니다. trAIlblazers 팀은 이를 다음과 같이 활용합니다.

*   **태그 생성:** 사전 정의된 목록에서 "Adventure" 또는 "Beach"와 같은 카테고리를 제안합니다.
*   **댓글 검토:** 게시되기 전에 댓글을 "Safe" 또는 "Harmful"로 분류합니다.

## 미디어 접근성

편집기는 Markdown의 "어려운 부분"을 자동화합니다. 이미지를 드롭하면 [Prompt API (멀티모달 입력 사용)](https://developers.google.com/ml-kit/genai/prompt/android/get-started)가 픽셀을 분석하여 접근 가능한 alt-text와 유익한 캡션을 생성합니다.

## 작성 및 재작성

[**Writer**](https://developer.chrome.com/docs/ai/writer-api) 및 [**Rewriter API**](https://developer.chrome.com/docs/ai/rewriter-api)를 사용하면 사용자는 글머리 기호를 완전한 단락으로 확장하고, 단락의 톤을 "더 캐주얼하게" 또는 "더 짧게" 변경할 수 있습니다.

## 원활한 번역

[**Translator API**](https://developer.chrome.com/docs/ai/translator-api)를 사용하면 영어로 콘텐츠를 작성하고 스페인어 또는 일본어 독자를 위해 즉시 번역할 수 있으며, 이는 원어민이 다듬을 수 있습니다.

![Translator API 작동 예시](https://developer.chrome.com/static/blog/build-new-features-using-built-in-ai-in-chrome-io2026/images/translator-api.png)

Translator API를 사용하여 영어로 콘텐츠를 작성하고 스페인어 및 일본어로 즉시 번역하세요.

# 실제 성공 사례

많은 파트너들이 이미 이러한 API를 프로덕션 환경에서 사용하고 있습니다. 주목할 만한 예시는 다음과 같습니다.

*   **Drupal:** CKEditor 내에서 SEO 태그 생성을 위해 Summarizer API를 사용합니다.
*   **Yahoo! Japan:** 커뮤니티 댓글 검토를 위해 Prompt API를 사용합니다.
*   **Trip.com:** AI 개요를 통해 복잡한 항공편 예약 옵션을 고객이 탐색하도록 돕습니다.
