# Chrome at Google I/O 2026 신기능

Google I/O 2026이 마무리되었습니다! Chrome 관련 발표 내용은 [Chrome at Google I/O 2026 웹사이트](https://io.google/2026/explore/?focus_areas=Chrome)에서 모두 확인하실 수 있습니다.

Chrome for Developers에서는 다음과 같은 발표 내용에 대해 자세히 알아볼 수 있습니다:

- [WebMCP](https://developer.chrome.com/docs/ai/webmcp)
- [Modern Web Guidance](https://developer.chrome.com/docs/modern-web-guidance)
- [Chrome DevTools for agents](https://developer.chrome.com/docs/devtools/agents)
- [HTML-in-Canvas](https://developer.chrome.com/blog/html-in-canvas-origin-trial)
- [Prompt API](https://developer.chrome.com/docs/ai/prompt-api)

시청해 주셔서 감사합니다. 내년 Google I/O 2027에서 다시 뵙겠습니다!
