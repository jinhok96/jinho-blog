Anthropic은 Claude Code를 활용하여 대규모 코드 마이그레이션을 6단계에 걸쳐 효율적으로 수행합니다.

- **1단계: 규칙서, 의존성 맵, 격차 인벤토리 생성:** 코드 번역 규칙, 파일 간 의존성, 리팩토링이 필요한 부분 등을 명확히 정의합니다. Claude Code는 규칙서 작성을 돕고, 복잡한 의존성 맵을 자동으로 생성합니다.
- **2단계: 규칙 테스트:** 소규모 코드로 규칙서의 유효성을 검증하고 잠재적인 문제를 미리 발견하여 전체 마이그레이션의 시행착오를 줄입니다.
- **3단계: 전체 코드 번역:** 정의된 규칙과 자동화된 에이전트를 사용하여 코드를 일괄적으로 번역하고, 컴파일 오류 및 잠재적 문제를 주석으로 표시하여 다음 단계를 준비합니다.

- **4-6단계: 컴파일, 실행, 동작 일치:** 번역된 코드를 컴파일하고, 테스트를 실행하여 발생하는 오류 및 충돌을 수정합니다. 최종적으로 원본 코드와 마이그레이션된 코드의 동작 일치 여부를 검증합니다. Claude Code는 이 과정에서 발생하는 반복적인 수정 작업을 자동화하고, 결과 기반으로 규칙서를 개선하는 데 활용됩니다.
