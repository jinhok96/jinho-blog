> 원문: [An update on recent Claude Code quality reports](https://www.anthropic.com/engineering/april-23-postmortem) — Anthropic Engineering
>
> 이 글은 원문의 핵심 내용을 요약한 것입니다. 저작권은 원문 저자에게 있으며, 자세한 내용은 원문에서 확인하세요.

- Claude Code의 기본 추론 노력을 `high`에서 `medium`으로 변경한 것이 코딩 품질을 저하시켜 `high`로 복구했습니다.
- 세션이 1시간 이상 유휴 상태일 때 Claude의 이전 추론 기록을 지우는 캐싱 최적화 버그로 인해 Claude가 잊어버리거나 반복하는 현상이 발생했으나 수정되었습니다.
- Claude Opus 4.7의 과도한 장황함을 줄이기 위한 시스템 프롬프트 변경이 코딩 품질에 부정적인 영향을 미쳐 롤백되었습니다.

이러한 문제는 API가 아닌 Claude Code, Claude Agent SDK 및 Claude Cowork에 영향을 미쳤으며, 4월 20일(v2.1.116)에 모두 해결되었습니다.

자세한 내용은 원문에서 확인하세요.
