Claude Design이 이제 프로젝트 전반에 걸쳐 디자인 시스템을 일관되게 유지하고, Claude Code와 유연하게 연동되며, 캔버스에서 직접 편집할 수 있게 되었습니다. 또한, 사용자가 이미 사용 중인 다양한 도구와 연결 기능이 강화되었습니다.

주요 변경 사항은 다음과 같습니다.

-   **디자인 시스템 컴포넌트 기반 구축**: GitHub 레포지토리, 디자인 파일, 또는 원시 업로드로부터 디자인 시스템을 가져와 Claude가 이를 기반으로 작업하고, 결과물을 디자인 시스템에 맞춰 자동으로 검사하고 수정합니다.
-   **디자인과 코드 간의 원활한 전환**: Claude Design과 Claude Code 간의 작업 전환이 더욱 쉬워졌으며, `/design-sync` 및 `/design` 명령어를 통해 디자인 시스템 동기화, 코드 기반 디자인 생성 및 편집, 라이브 프로토타입 제작 등이 가능합니다.
-   **직접 편집 기능 및 안정성 향상**: 새로운 편집기는 요소의 드래그, 크기 조절, 정렬을 위한 풍부한 레이아웃 제어 기능을 제공하며, 수백 가지의 안정성 개선으로 실사용 환경에서도 안정적인 경험을 제공합니다.
-   **외부 도구와의 확장된 연동**: PDF 및 PowerPoint로의 안정적인 내보내기 기능과 더불어 Adobe, Replit, Vercel, Wix, Canva, Miro 등 다양한 도구와의 연동이 가능해져 기존 워크플로우에 seamless하게 통합됩니다.

Claude Design은 Claude Pro, Max, Team, Enterprise 플랜에서 베타 버전으로 제공됩니다. 자세한 내용은 원문에서 확인하세요.
