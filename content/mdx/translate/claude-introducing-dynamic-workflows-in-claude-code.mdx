- Claude Code에 새롭게 도입된 동적 워크플로우는 수십에서 수백 개의 병렬 하위 에이전트를 실행하여 복잡한 코딩 작업을 엔드투엔드로 처리합니다.
- 이 기능은 코드베이스 전반의 버그 탐색, 대규모 마이그레이션, 중요한 코드 검증 등 복잡한 문제 해결에 효과적입니다.
- 사용자는 "워크플로우 생성"을 요청하거나 `ultracode` 설정을 활성화하여 동적 워크플로우를 시작할 수 있습니다.
- 동적 워크플로우는 Zig에서 Rust로 Bun을 포팅하는 사례와 같이 대규모 프로젝트에 적용되어 효율성을 입증했습니다.
- 이 기능은 Max, Team, Enterprise 플랜 및 Claude API 등 다양한 환경에서 사용할 수 있으며, 사용량이 많은 점을 고려하여 시작 시 주의가 필요합니다.
