> 원문: [Introducing the Claude apps gateway for Amazon Bedrock and Google Cloud](https://claude.com/blog/introducing-the-claude-apps-gateway) — Claude Blog
> 이 글은 원문의 핵심 내용을 요약한 것입니다. 저작권은 원문 저자에게 있으며, 자세한 내용은 원문에서 확인하세요.

- Claude Apps Gateway는 Amazon Bedrock 및 Google Cloud에서 Claude Code를 위한 기업용 SSO 로그인, 중앙 집중식 정책 적용, 역할 기반 액세스, 사용자별 비용 추적 기능을 제공합니다.
- 게이트웨이는 단일 상태 비저장(stateless) 컨테이너로 실행되며, OIDC 호환 IdP를 통해 개발자를 인증하고, 관리 설정을 배포하며, 사용자별 사용량 데이터를 수집합니다.
- 주요 기능으로는 OIDC 기반 ID 관리, 중앙 집중식 정책 설정, OTLP를 통한 텔레메트리 전송, Claude API, Amazon Bedrock, Google Cloud로의 추론 라우팅, 조직/그룹/사용자별 비용 상한 설정 등이 있습니다.
- 클라이언트와 게이트웨이가 함께 빌드되어 로그인 시 관리 설정 자동 적용 및 일관된 정책이 시행됩니다.
- 지금 바로 Claude Code CLI 바이너리를 다운로드하고 `gateway.yaml` 파일을 구성하여 게이트웨이를 배포할 수 있습니다.
