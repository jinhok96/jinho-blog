Claude Code는 CLAUDE.md 파일, 규칙, 스킬, 하위 에이전트, 훅, 출력 스타일 등 다양한 기능을 통해 Claude의 동작을 세밀하게 제어할 수 있도록 합니다.

- **규칙 (Rules)**: `.claude/rules/`에 위치하며, 특정 코드베이스의 제약 조건이나 컨벤션을 정의합니다. `paths` 필드를 사용하여 특정 경로에만 적용되도록 스코핑할 수 있습니다.
```
---
paths:
  - "src/api/**"
  - "**/*.handler.ts"
---
All API handlers must validate input with Zod before processing.
```
- **스킬 (Skills)**: `.claude/skills/`에 폴더 형태로 존재하며, 지침, 스크립트, 리소스를 동적으로 로드합니다. `SKILL.md` 파일에 이름, 설명, 본문이 포함되며, `/code-review`와 같은 슬래시 명령어로 트리거됩니다.
- **하위 에이전트 (Subagents)**: `.claude/agents/`에 markdown 파일로 정의되며, 특정 부수 작업을 위한 격리된 어시스턴트입니다. 자체 컨텍스트 창에서 실행되며, 최종 메시지만 메인 세션으로 반환됩니다.
- **훅 (Hooks)**: 파일 편집, 도구 호출 등 Claude의 특정 이벤트에 반응하는 사용자 정의 명령, HTTP 엔드포인트, LLM 프롬프트입니다. `settings.json` 등에 등록되며, 낮은 컨텍스트 비용으로 결정론적인 제어를 제공합니다.
