- AWS, Google Cloud, Microsoft Foundry를 통해 Claude Desktop을 사용하는 조직에 이제 채팅, Claude Cowork, Claude Code 기능을 모두 포함한 전체 Claude Desktop 경험이 제공됩니다.
- IT 팀은 자체 환경 내에서 추론을 유지하면서 SSO, MDM 정책 템플릿, 오프라인 설치 옵션, 기기에서 완전히 실행되는 M365 커넥터를 사용하여 조직 전체에 Claude Desktop을 배포할 수 있습니다.
- 추론은 구성된 리전에서 조직의 클라우드에서 실행되며, 대화 기록은 로컬에 저장됩니다.
- 데이터 커넥터가 접근하는 엔드포인트와 Anthropic이 받는 집계된 원격 분석을 사용자가 제어합니다.
