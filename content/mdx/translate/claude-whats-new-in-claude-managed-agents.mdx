Claude Managed Agents에 다음과 같은 두 가지 새로운 기능이 도입되었습니다.

-   **예약 실행 기능:** 에이전트가 정해진 시간에 자동으로 실행되어 반복적인 작업을 처리할 수 있게 되었습니다. 이를 통해 야간 데이터 동기화, 주간 규정 준수 검사 등 정기적인 업무 자동화가 가능합니다.
-   **Vault 환경 변수 저장:** CLI 도구나 인증된 서비스에 안전하게 접근하기 위해 환경 변수를 Vault에 저장할 수 있게 되었습니다. 에이전트는 API 키를 직접 노출하지 않고도 Vault에 저장된 변수를 활용하여 인증된 요청을 보낼 수 있습니다.

이러한 기능들은 현재 공개 베타 버전으로 Claude Platform에서 사용할 수 있습니다.
