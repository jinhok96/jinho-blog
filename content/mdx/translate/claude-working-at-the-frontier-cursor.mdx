Cursor 엔지니어 Nate Schmidt는 Claude Fable 5가 개발자들이 직면하는 가장 어려운 문제들을 해결하는 데 탁월한 성능을 보여준다고 말합니다. Cursor는 자체 벤치마크인 CursorBench를 개발하여 모델들이 실제 개발 시나리오에서 얼마나 잘 작동하는지 평가하는데, Claude Fable 5는 이 벤치마크에서 새로운 최고 기록을 세웠습니다.

- **CursorBench를 통한 성능 입증:** Claude Fable 5는 스택 트레이스 분석, 잘못된 모듈 식별 등 모호하고 복잡한 개발 문제에 대한 평가에서 72.9%의 최고 점수를 기록했습니다.
- **복잡한 문제 해결 능력:** Schmidt는 Claude Fable 5가 개발자의 의도를 스스로 파악하고, 전역적인 관점에서 문제를 해결하며, 반복적인 지시 없이도 복잡한 리팩터링이나 엣지 케이스 추론을 성공적으로 수행한다고 설명합니다.
- **우주선 착륙 시뮬레이션:** 과거 Claude Opus가 장시간 실패했던 달 착륙 시뮬레이션에서 Claude Fable 5는 초기 궤도 비행 및 원격 측정을 통한 정보 수집 후 성공적으로 착륙하는 전역적 사고 능력을 보여주었습니다.
- **전략적 모델 활용:** Cursor 팀은 비용 효율성과 성능을 고려하여, 일반적인 작업에는 더 빠르고 저렴한 모델을 사용하고, "전역 최적점"을 찾아야 하는 복잡한 문제에 Claude Fable 5를 투입하는 방식으로 활용하고 있습니다.
