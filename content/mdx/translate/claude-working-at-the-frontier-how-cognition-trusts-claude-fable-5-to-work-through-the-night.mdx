Cognition은 개발 초기부터 Claude 모델을 활용하여 AI 소프트웨어 엔지니어 Devin을 구축해 왔습니다. 이전 모델들은 몇 분에서 한 시간 정도만 on-task 상태를 유지할 수 있었으나, Claude Fable 5는 이러한 한계를 극복했습니다.

- Claude Fable 5는 이전 모델 대비 Frontier Code 벤치마크에서 3배 높은 점수를 기록하며, 실제 코드베이스 환경에서 더 뛰어난 성능을 보였습니다.
- 이 모델은 최대 8시간 동안 스스로 작업을 수행하며 꾸준히 진전을 이루는 능력을 보여주어, 엔지니어들이 밤샘 작업을 맡길 수 있을 정도의 안정성을 확보했습니다.
- Claude Fable 5는 복잡하고 잡음이 많은 컨텍스트에서도 명확한 사고를 유지하며, 내부 디버깅 도구를 효과적으로 사용하여 로그를 분석하고 결론을 도출합니다.
- 이러한 개선 덕분에 Devin은 이제 Slack 채널을 모니터링하거나 프로덕션 환경의 문제를 스스로 감지하고 수정하는 등, 실제 엔지니어처럼 능동적인 작업을 수행할 수 있게 되었습니다.
- Cognition은 Claude Fable 5를 통해 클라우드에서 몇 시간씩 실행되는 AI 에이전트에 대한 초기 비전을 실현하고 있으며, 이는 향후 엔지니어링 팀의 기본 작업 방식으로 자리 잡을 것으로 예상됩니다.
