Next.js 16.2는 Turbopack을 기본 번들러로 채택한 이후 성능 개선, 버그 수정 및 호환성 향상에 집중했습니다. 주요 변경 사항은 다음과 같습니다.

- **Server Fast Refresh**: 개발 중 서버 코드 핫 리로딩이 훨씬 효율적으로 개선되었습니다.
- **Web Worker Origin**: WASM 라이브러리 지원이 강화되어 Worker 내에서 `fetch()` 사용이 용이해졌습니다.
- **Subresource Integrity Support**: JavaScript 파일에 대한 Subresource Integrity (SRI)를 지원하여 파일 무결성을 검증합니다.
- **Tree Shaking of Dynamic Imports**: 동적 `import()`에서도 사용되지 않는 export를 제거합니다.
- **Inline Loader Configuration**: `import attributes`를 통해 개별 import에 대한 로더 설정을 지원합니다.
- **Lightning CSS Configuration**: 실험적인 `lightningCssFeatures` 옵션으로 CSS 트랜스파일링 기능을 제어할 수 있습니다.
- **Log Filtering**: `ignoreIssue` 옵션으로 불필요한 로그나 경고를 억제할 수 있습니다.
- **`postcss.config.ts` Support**: TypeScript 기반의 PostCSS 설정을 지원합니다.
- **Performance Improvements and Bug Fixes**: 200개 이상의 변경 및 버그 수정으로 안정성과 호환성이 향상되었습니다.
