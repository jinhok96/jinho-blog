Next.js 16.3은 AI 기반 개발을 지원하기 위한 다양한 개선 사항을 포함하고 있습니다. 주요 변경 사항은 다음과 같습니다.

- **AGENTS.md를 통한 문서 번들링**: AI 에이전트가 프로젝트의 최신 버전에 맞는 문서를 로컬에서 쉽게 참조할 수 있도록 `next dev` 명령어로 `AGENTS.md` 파일에 문서 포인터를 자동 업데이트합니다.
- **첫 파티 스킬 도입**: `next-dev-loop`, `next-cache-components-adoption`, `next-cache-components-optimizer`와 같은 새로운 스킬을 통해 AI 에이전트가 브라우저 조작, 캐시 컴포넌트 적용, 성능 최적화 등 다단계 워크플로우를 수행할 수 있도록 지원합니다.
- **React Introspection 지원 Agent Browser**: `agent-browser` CLI에 React DevTools Introspection 기능이 추가되어, AI 에이전트가 React 컴포넌트 트리 탐색, 상태 검사, 렌더링 프로파일링 등을 수행할 수 있게 됩니다.
- **실행 가능한 에러 메시지**: 에러 발생 시, 제안되는 수정 사항에 대한 'Copy as prompt' 버튼과 각 규칙에 맞는 상세 문서를 제공하여 AI 에이전트가 더 효과적으로 문제를 해결하도록 돕습니다.
