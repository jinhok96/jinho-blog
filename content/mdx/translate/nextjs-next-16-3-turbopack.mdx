Next.js 16.3 릴리스와 함께 Turbopack은 다음과 같은 주요 개선 사항을 제공합니다.

- 개발 서버 메모리 사용량을 최대 90%까지 줄여 장기 실행 개발 세션에서 시스템 메모리 부담을 완화합니다.
- 빌드 시간을 단축하기 위해 이전 컴파일 결과를 디스크에 영구적으로 저장하는 파일 시스템 캐시를 지원합니다.
- Rust로 작성된 React 컴파일러를 실험적으로 지원하여 대규모 React 애플리케이션의 컴파일 속도를 20-50% 향상시킵니다.
- Vite와 호환되는 `import.meta.glob` API를 지원하여 동적으로 모듈을 가져올 수 있습니다.
- HMR(Hot Module Replacement) 및 개발 서버 시작 속도를 개선하여 전반적인 개발 경험을 향상시킵니다.

```rust
// Rust React Compiler 예시 (실험적 기능)
// --turbo --experimental-turbopack --turbopack-rust-react-compiler
```
