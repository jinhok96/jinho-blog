Next.js는 보안 강화 및 투명성 확보를 위해 새로운 보안 릴리스 프로그램을 도입합니다.

-   **정기적인 보안 릴리스 발표**: 앞으로 매달 Next.js 블로그를 통해 예정된 보안 릴리스에 대한 사전 공지를 게시합니다. 각 공지에는 예상 릴리스 일정과 예상되는 최고 심각도가 포함됩니다.
-   **긴급 패치는 계속 제공**: 즉각적인 조치가 필요하거나 이미 악용되고 있는 취약점의 경우, 비정기적인 긴급 패치를 계속 제공합니다.
-   **2026년 7월 20일 첫 예정 릴리스**: Next.js 16.2 및 15.5 버전을 대상으로 4개의 높은 심각도와 5개의 중간 심각도 취약점을 해결하는 첫 번째 예정된 보안 릴리스가 2026년 7월 20일에 출시될 예정입니다.
-   **보안 프로그램 참여 독려**: Vercel의 Open Source Bug Bounty 프로그램을 통해 Next.js 보안에 기여할 수 있습니다.
