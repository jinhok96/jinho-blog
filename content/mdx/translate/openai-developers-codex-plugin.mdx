OpenAI 개발자 플러그인은 Codex 환경에서 OpenAI Platform 연동 및 API 설정 가이드를 제공하여 AI 애플리케이션과 에이전트 개발을 돕습니다. Claude Code 및 Cursor용으로도 제공되며, 휴대 가능한 개발자 스킬과 OpenAI Docs MCP 서버를 포함합니다.

주요 기능은 다음과 같습니다.

*   **OpenAI API Platform 연동**: Codex를 [OpenAI API Platform](https://platform.openai.com/)에 연결합니다.
*   **OpenAI Docs MCP 활용**: Claude Code 또는 Cursor에서 최신 OpenAI 문서를 사용합니다.
*   **API 키 설정**: Codex에서 프로젝트 API 키를 생성, 저장, 연결하거나 Claude Code 및 Cursor에서 로컬 `OPENAI_API_KEY` 설정을 안내받습니다.
*   **Agents SDK**: 아이디어, 저장소 또는 이전 Codex 스레드에서 OpenAI Agents SDK 앱을 빌드하고 배포합니다.
*   **문제 해결**: 일반적인 OpenAI API 오류를 식별하고 적절한 다음 단계를 안내합니다.

Codex 사용자는 플러그인 설치 전 [Codex 앱 다운로드](https://developers.openai.com/codex/app#getting-started) 및 [Codex 퀵스타트](https://developers.openai.com/codex/quickstart)를 따르는 것이 좋습니다. 설치 후에는 OpenAI API 생성, API 키 설정, API 문제 진단 등 OpenAI Platform 연동이 필요한 작업에 자동으로 활용됩니다.

Claude Code와 Cursor는 플러그인의 스킬과 번들된 Docs MCP 서버를 통해 OpenAI API, 모델, Agents SDK, ChatGPT Apps 관련 안내를 받을 수 있습니다.

다음과 같은 상황에서 플러그인을 활용할 수 있습니다.

*   OpenAI API를 사용하는 앱 또는 에이전트 빌드
*   개발 중인 앱에 대한 OpenAI API 액세스 설정
*   일반적인 OpenAI API 오류 진단 및 해결 방안 모색
