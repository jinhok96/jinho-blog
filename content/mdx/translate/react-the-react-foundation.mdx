The React Foundation has officially launched, hosted by the Linux Foundation.

* * *

[지난 10월](https://react.dev/blog/2025/10/07/introducing-the-react-foundation), React Foundation 설립 의사를 발표했습니다. 오늘, React Foundation이 공식적으로 출범했음을 기쁜 마음으로 공유합니다.

React, React Native 및 JSX와 같은 지원 프로젝트는 더 이상 Meta가 소유하지 않습니다. 이제 Linux Foundation에 의해 호스팅되는 독립적인 재단인 React Foundation이 소유합니다. 자세한 내용은 [Linux Foundation 보도 자료](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-react-foundation)에서 확인할 수 있습니다.

## Founding Members

React Foundation에는 **Amazon**, **Callstack**, **Expo**, **Huawei**, **Meta**, **Microsoft**, **Software Mansion**, **Vercel** 등 8개의 Platinum 창립 멤버가 있습니다. **Huawei**는 [10월 발표](https://react.dev/blog/2025/10/07/introducing-the-react-foundation) 이후 합류했습니다. React Foundation은 각 회원사의 대표로 구성된 이사회에 의해 운영되며, [Seth Webster](https://sethwebster.com/)가 이그제큐티브 디렉터를 맡습니다.

## New Provisional Leadership Council

React의 기술 거버넌스는 React Foundation 이사회와 항상 독립적으로 운영될 것입니다. React의 기술적 방향은 계속해서 React에 기여하고 유지보수하는 사람들에 의해 설정될 것입니다. 이 구조를 결정하기 위해 임시 리더십 위원회를 구성했습니다. 몇 달 안에 업데이트를 공유할 예정입니다.

## Next Steps

전환을 완료하기 위한 작업이 아직 남아 있습니다. 향후 몇 달 안에 다음을 진행할 것입니다.

*   React의 기술 거버넌스 구조 최종 확정
*   저장소, 웹사이트 및 기타 인프라를 React Foundation으로 이전
*   React 생태계를 지원하기 위한 프로그램 탐색
*   차기 React Conf 계획 착수

이 작업이 진행됨에 따라 업데이트를 공유할 것입니다.

## Thank You

지난 10년간 React를 발전시켜 온 수천 명의 기여자 없이는 이 모든 것이 불가능했을 것입니다. 창립 멤버, 풀 리퀘스트를 열거나, 이슈를 제출하거나, 다른 사람이 React를 배우는 것을 도운 모든 기여자, 그리고 매일 React로 빌드하는 수백만 명의 개발자 여러분께 감사드립니다. React Foundation은 이 커뮤니티 덕분에 존재하며, 우리는 함께 미래를 만들어갈 것입니다.
