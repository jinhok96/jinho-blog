> 원문: [The React Foundation: A New Home for React Hosted by the Linux Foundation](https://react.dev/blog/2026/02/24/the-react-foundation) — React Blog

The React Foundation has officially launched, hosted by the Linux Foundation.

[지난 10월](https://react.dev/blog/2025/10/07/introducing-the-react-foundation), React Foundation 설립 의사를 발표했습니다. 오늘, React Foundation이 공식적으로 출범했음을 기쁘게 공유합니다.

React, React Native 및 JSX와 같은 지원 프로젝트는 더 이상 Meta의 소유가 아니며, 이제 리눅스 재단에서 호스팅하는 독립적인 재단인 React Foundation의 소유가 됩니다. 자세한 내용은 [리눅스 재단의 보도 자료](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-react-foundation)에서 확인할 수 있습니다.

## Founding Members[](#founding-members "Link for Founding Members ")

React Foundation에는 8개의 플래티넘 창립 멤버가 있습니다: **Amazon**, **Callstack**, **Expo**, **Huawei**, **Meta**, **Microsoft**, **Software Mansion**, **Vercel**. **Huawei**는 [지난 10월 발표](https://react.dev/blog/2025/10/07/introducing-the-react-foundation) 이후 합류했습니다. React Foundation은 각 회원사의 대표로 구성된 이사회에 의해 운영되며, [Seth Webster](https://sethwebster.com/)가 이사로 활동합니다.

## New Provisional Leadership Council[](#new-provisional-leadership-council "Link for New Provisional Leadership Council ")

React의 기술 거버넌스는 React Foundation 이사회와 항상 독립적으로 유지될 것입니다. React의 기술적 방향은 React에 기여하고 유지보수하는 사람들에 의해 계속 설정될 것입니다. 우리는 이 구조를 결정하기 위해 임시 리더십 위원회를 구성했습니다. 몇 달 안에 업데이트를 공유할 예정입니다.

## Next Steps[](#next-steps "Link for Next Steps ")

이전을 완료하기 위한 작업이 아직 남아 있습니다. 향후 몇 달 동안 다음과 같은 작업을 진행할 예정입니다:

*   React의 기술 거버넌스 구조 확정
*   리포지토리, 웹사이트 및 기타 인프라를 React Foundation으로 이전
*   React 생태계를 지원하기 위한 프로그램 탐색
*   차기 React Conf 계획 착수

이 작업이 진행됨에 따라 업데이트를 공유할 것입니다.

## Thank You[](#thank-you "Link for Thank You ")

지난 10년간 React를 만들어 온 수천 명의 기여자들이 없었다면 이 모든 것은 불가능했을 것입니다. 창립 멤버, 풀 리퀘스트를 열거나, 이슈를 제출하거나, 다른 사람이 React를 배우도록 도운 모든 기여자, 그리고 매일 React로 구축하는 수백만 명의 개발자에게 감사드립니다. React Foundation은 이 커뮤니티 덕분에 존재하며, 우리는 함께 미래를 만들어 나가기를 기대합니다.
