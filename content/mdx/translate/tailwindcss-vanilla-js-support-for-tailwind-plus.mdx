> 원문: [Vanilla JavaScript support for Tailwind Plus](https://tailwindcss.com/blog/vanilla-js-support-for-tailwind-plus) — Tailwind CSS Blog
> 이 글은 원문의 핵심 내용을 요약한 것입니다. 저작권은 원문 저자에게 있으며, 자세한 내용은 원문에서 확인하세요.

- Tailwind Plus의 모든 UI 블록(다이얼로그, 드롭다운, 커맨드 팔레트 등)이 이제 JavaScript 프레임워크 없이 바로 사용 가능합니다.
- `@tailwindplus/elements` 라이브러리는 HTML, CSS 유틸리티 클래스만으로 인터랙티브 UI를 구축할 수 있는 헤드리스 커스텀 엘리먼트를 제공합니다.
- 이 커스텀 엘리먼트는 `<script>` 태그로 어디서든 작동하며, ARIA 속성 관리, 포커스 처리, 키보드 지원 등 복잡한 동작을 내장합니다.
- 최신 웹 플랫폼 기능을 활용하여 가볍고 네이티브에 가까운 경험을 제공하며, Tailwind CSS v4.0이 지원하는 모든 브라우저에서 작동합니다.
- Svelte, Rails, React 등 다양한 프레임워크에서 HTML의 범용성을 바탕으로 Tailwind Plus UI 블록을 쉽게 통합할 수 있습니다.
