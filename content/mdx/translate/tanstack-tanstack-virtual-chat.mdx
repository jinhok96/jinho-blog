채팅 UI는 더 이상 단순한 리스트가 아니며, TanStack Virtual은 이를 위한 새로운 기능을 추가했다.

- **역방향 무한 스크롤:** 새로운 메시지가 끝에 추가되고, 오래된 메시지는 앞에 추가되는 채팅 UI의 특성에 맞춰 `anchorTo: 'end'` 옵션을 통해 목록의 끝을 기준으로 스크롤을 유지한다.
- **조건부 팔로우:** 사용자가 과거 메시지를 읽고 있을 경우, 새 메시지가 도착해도 스크롤 위치를 강제로 바꾸지 않고, 최신 메시지를 보고 있을 때는 자동으로 따라가도록 `followOnAppend` 옵션으로 제어한다.
- **스트리밍 메시지 처리:** 토큰별로 메시지가 늘어나는 경우에도 스크롤 위치를 유지하며, `anchorTo: 'end'` 옵션으로 목록 끝에 고정된 상태를 유지한다.
- **스크롤 위치 유지:** `anchorTo: 'end'` 옵션은 데이터 변경 시에도 현재 보이는 항목을 기억하고, 데이터 변경 후에도 동일한 시각적 위치를 유지하도록 스크롤 오프셋을 조정한다.
- **"최신으로 이동" 버튼 지원:** `scrollToEnd()`와 같은 헬퍼 함수를 제공하여 사용자가 최신 메시지로 쉽게 이동할 수 있도록 한다.

TanStack Virtual은 채팅 UI에 필요한 스크롤 로직을 추상화하여 개발자가 UI 컴포넌트 구현에 집중할 수 있도록 돕는다.
