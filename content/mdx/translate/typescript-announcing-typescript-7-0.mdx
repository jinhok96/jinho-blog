TypeScript 7.0이 발표되었습니다. 이번 릴리스는 Go 언어로 재작성된 네이티브 포트로, 이전 버전에 비해 10배 이상 빨라진 성능을 자랑합니다.

-   **핵심 성능 향상:** TypeScript 7.0은 Go 언어로 작성된 네이티브 포트를 통해 전체 빌드 시간을 최대 11.9배 단축하고 메모리 사용량을 최대 26%까지 줄였습니다.
-   **향상된 개발 경험:** 에디터 로딩, 코드 탐색, 자동 완성 등 개발 과정 전반의 속도가 크게 향상되어 더욱 빠르고 부드러운 개발이 가능합니다.
-   **병렬 처리 및 제어 기능:** `--checkers`, `--builders`, `--singleThreaded`와 같은 새로운 플래그를 통해 병렬 처리 동작을 세밀하게 제어할 수 있어, 대규모 코드베이스와 다양한 환경에 최적화된 성능을 제공합니다.
-   **개선된 `--watch` 모드:** Parcel 번들러의 파일 감시자를 기반으로 새롭게 구축된 `--watch` 모드는 더욱 효율적이고 안정적인 파일 감시 기능을 제공합니다.
-   **JavaScript 지원 강화:** TypeScript 7.0은 JavaScript 파일 분석 방식을 개선하여 TypeScript `.ts` 파일 분석과의 일관성을 높였으며, 일부 JSDoc 태그 및 패턴 지원을 중단했습니다.
