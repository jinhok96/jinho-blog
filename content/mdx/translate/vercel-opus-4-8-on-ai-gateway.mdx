- Claude Opus 4.8 모델이 Vercel AI Gateway에서 제공됩니다.
- 이 모델은 장기적인 에이전트 실행에 최적화되어 복잡한 다단계 코딩 작업 및 사람의 개입 없이도 수정 가능한 리팩토링을 처리합니다.
- 문서 초안 작성, 데이터 분석, 프레젠테이션 구축 등 지식 작업에서 더 명확하고 간결한 결과물을 생성합니다.
- AI Gateway는 통합 API, 사용량 및 비용 추적, 재시도 및 장애 조치 구성, 성능 최적화 기능을 제공하며, 제공업체 가격에 마크업 없이 Zero Data Retention, 동적 프로바이더 정렬 등의 기능을 지원합니다.

```
import { streamText } from 'ai';
const result = streamText({
  model: 'anthropic/claude-opus-4.8',
  prompt: 'Find and fix the root cause of these intermittent test failures.',
  providerOptions: {
    anthropic: {
      thinking: { type: 'adaptive' },
      effort: 'high',
    },
  },
});
```
