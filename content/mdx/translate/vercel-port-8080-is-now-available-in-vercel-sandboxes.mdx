Vercel 샌드박스에서 이제 8080 포트를 열어 사용할 수 있습니다. 이전에는 컨트롤러 포트로 사용되었던 8080 포트가 23456으로 이전되면서, 개발자는 Vercel 샌드박스 내에서 직접 애플리케이션을 실행하고 테스트하기 위해 8080 포트를 자유롭게 활용할 수 있게 되었습니다.

- Vercel 샌드박스에서 8080 포트가 공식적으로 지원됩니다.
- 기존 컨트롤러 포트(8080)는 23456으로 이전되었습니다.
- 이를 통해 개발자는 샌드박스 환경에서 웹 서버 등을 8080 포트로 직접 실행할 수 있습니다.

```
import { Sandbox } from "@vercel/sandbox";const sandbox = await Sandbox.create({  ports: [8080],});await sandbox.runCommand({  cmd: "python3",  args: ["-m", "http.server", "8080", "--bind", "0.0.0.0"],  detached: true,});console.log(`url: ${sandbox.domain(8080)}`);
```
