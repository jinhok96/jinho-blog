Vercel Sandbox가 이제 Docker 설치 및 실행을 지원합니다. 이를 통해 에이전트는 호스트 시스템에 영향을 주지 않고 컨테이너를 빌드하고, 시스템 패키지를 설치하며, 파일을 수정할 수 있습니다.

- Vercel Sandbox에서 Docker를 설치하고 데몬을 시작하여 컨테이너화된 애플리케이션을 서비스할 수 있습니다.
- Redis 또는 Postgres와 같은 컨테이너화된 서비스를 테스트 종속성으로 실행하는 데 유용합니다.
- 배포 전 컨테이너 이미지를 검증하거나 컨테이너에서 서비스되는 애플리케이션을 미리 볼 수 있습니다.
- 영구 샌드박스와 결합하면 Docker 설치 및 가져온 이미지가 세션 간에 유지됩니다.
- Docker 지원 외에도 FUSE 파일 시스템 드라이버 및 VPN 클라이언트를 지원하여 활용 범위를 확장합니다.
