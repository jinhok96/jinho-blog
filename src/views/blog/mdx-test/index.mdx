## MDX 테스트

안녕하세요! 이것은 MDX 테스트 포스트입니다.

### 코드 하이라이팅 테스트

```tsx
import type { NextConfig } from 'next';

const nextConfig: NextConfig = {
  pageExtensions: ['js', 'jsx', 'ts', 'tsx', 'md', 'mdx'],
};

export default nextConfig;
```

### 리스트 테스트

#### 순서 없는 리스트

- 첫 번째 항목
- 두 번째 항목
- 세 번째 항목
  - 중첩된 항목 1
  - 중첩된 항목 2

#### 순서 있는 리스트

1. 첫 번째 단계
2. 두 번째 단계
3. 세 번째 단계

### 테이블 테스트

| 기능  | 설명                     | 상태    |
| ----- | ------------------------ | ------- |
| MDX   | Markdown + JSX           | ✅ 완료 |
| Prism | 코드 하이라이팅          | ✅ 완료 |
| GFM   | GitHub Flavored Markdown | ✅ 완료 |
